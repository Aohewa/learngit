
working directory, 工作区
例如，learngit, 电脑中的一个目录

===========================

repository, 版本库
工作区的隐藏目录 .git
例如，learngit 目录中的 .git 目录

===========================

.git 目录中存放着很多文件

index文件叫做暂存区
HEAD文件叫做指针，指针指向master分支

============================

前面把文件添加到Git时，分了两步

---------------
1、git add <file>
把文件添加到暂存区

2、git commit -m <message>
把暂存区中的所有内容提交到当前分支
---------------

创建Git版本库时，Git自动创建了唯一的master分支
git commit 命令，就是往master分支上提交修改
