
先创建远程库，然后，从远程库克隆到本地仓库

============================

本地库运行命令
----------------

$ git colone 远程库地址

----------------

多人协作开发

每个人各自克隆一份远程库就可以了

----------------

GitHub支持多个协议，给出的地址不止一个
SSH、https

https速度慢，每次推送都需要口令，但是有些公司只开放http端口
无法使用ssh

=============================
小结
	克隆一个仓库，必须知道仓库地址
		git clone 仓库地址
	Git支持多种协议，ssh支持的原生git协议速度最快
